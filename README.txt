-------------------------------------
              MINI MATH
-------------------------------------

An educational program that randomly
generates simple math questions.

-------------------------------------

"Final Files"
     This folder contains all of the
     finished files, including the
     Setup.jar file which will
     install the Mini Math software.


"Program Contents"
     This folder contains all of the
     raw files that make up the final
     program, including .java and
     .class files, images and fonts.

-------------------------------------
